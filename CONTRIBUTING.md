# Contributing to Wallet Recovery Tool

## Setup

1. Clone the repo
2. Install dependencies: `pip3 install mnemonic eth-account web3`
3. Run the tool: `python3 wallet-recovery.py --help`

## Pull Request Guidelines

- Keep changes focused and minimal
- Test with the standard test mnemonic before submitting
- Update the README if adding new features
- All code must be Python 3.8+ compatible

## Code Style

- Follow PEP 8
- Type hints required for function signatures
- Docstrings for public functions
- No external dependencies beyond those in requirements.txt

## Reporting Issues

- Include the exact command and error output
- Specify your Python version and OS
- Remove any sensitive data (real mnemonics, private keys)

## Questions?

Open a GitHub Discussion or issue.
