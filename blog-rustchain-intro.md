# RustChain: The Blockchain Where Vintage Hardware Outearns Modern Machines

## What Is RustChain?

RustChain is a DePIN (Decentralized Physical Infrastructure Network) that uses AI-powered hardware fingerprinting to verify real physical machines. Unlike proof-of-work or proof-of-stake, RustChain uses **Proof-of-Antiquity** — where older, vintage hardware earns MORE mining rewards than modern systems.

A PowerBook G4 from 2003 earns 2.5x more than a Threadripper. A Power Mac G5 earns 2.0x. This creates a unique economic incentive to preserve computing history.

## How It Works

RustChain validates miners through:

1. **Hardware Fingerprinting** — AI analyzes oscillator drift, thermal curves, and timing signatures unique to each physical machine
2. **Proof-of-Antiquity** — Older hardware receives higher epoch rewards, incentivizing vintage machine preservation
3. **Ed25519 Cryptography** — Native wallet system using Ed25519 keypairs for secure transfers

## Getting Started

Install the miner with one command:
```bash
curl -sL https://rustchain.org/install.sh | bash
```

Creating a wallet:
```bash
# Using the wallet CLI
python3 tools/rustchain_wallet_cli.py create --name my-wallet

# Or use the secure wallet GUI
python3 wallet/rustchain_wallet_secure.py
```
